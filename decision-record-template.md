# Decision Record Template

## Purpose

Use this template to save important WebReadyNow decisions so they are not lost in chat history.

---

## When to Create a Decision Record

Create a decision record for:

- SEO decisions
- New content or page decisions
- Product positioning decisions
- Product roadmap decisions
- Service offer decisions
- Safe claim changes
- Major agent recommendations
- Important strategy changes

---

## Where to Save Decision Records

| Decision Type | Save To |
|---|---|
| Research decisions | `09-agent-outputs/researcher/` |
| Product alignment decisions | `09-agent-outputs/product-alignment/` |
| Codebase audit findings | `09-agent-outputs/codebase-audits/` |
| SEO decisions | `09-agent-outputs/seo-decisions/` |
| Marketing and docs decisions | `09-agent-outputs/marketing-docs/` |
| Ops and delivery decisions | `09-agent-outputs/ops-delivery/` |
